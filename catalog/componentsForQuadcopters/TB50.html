Описание

Интеллектуальная батарея ТВ50 для Inspire 2 / Ronin 2. Обеспечивает в паре с другой батареей 25 минут полета с подвесом Zenmuse X5S, 27 минут с подвесом Zenmuse X4S. Батареи ТВ50 относятся к новому поколению интеллектуальных полетных батарей DJI, оснащенных новым блоком управления и отличающихся повышенной производительностью и плотностью.

Новый блок управления позволяет выполнять самостоятельный разогрев батарей при низких температурах. Система самостоятельного разогрева может быть активирована вручную и в автоматическом режиме. Информация о состоянии аккумуляторов и текущих изменениях с помощью блока управления передается пилоту через приложение DJI GO.

При использовании со стабилизатором Ronin 2 поддерживает горячую замену батарей для обеспечения беспрерывного съемочного процесса.
